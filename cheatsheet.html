Questions + Answers for HTML

Name the components of html elements properly

Start_tag, content, End_tag

Void elements

Void elements: elements with no content
Example: br/

Attributes

All HTML elements can have attributes. Attributes provide additional information about an element. They're always specified in the start tag. They usually come in name/value pairs: name="value" < a href = "http://bsp.com"> This is a link

DOCTYPE

< !DOCTYPE html >

Elements

meta info in head: <meta charset="UTF-8"> <link rel="stylesheet" href="CheatSheet2.css"> Comment: <!-- This would be a comment --> Title: title in a browser tab

CSS rule

selector - declaration

h1 {property:value; property:value;}

Combinators

descendant selector (space), The descendant selector matches all elements that are descendants of a specified element.
child selector (>), the child selector selects all elements that are the immediate children of a specified element
adjacent sibling selector (+), The adjacent sibling selector selects all elements that are the adjacent siblings of a specified element.
general sibling selector (~), The general sibling selector selects all elements that are siblings of a specified element.

Fonts

erif: fonts have small lines at the ends on some characters sans-serif: sans=without, no lines

font-family: should hold some font names as an option if the browser doesn't support the 1st font

Example:
p{
font-family: "Times New Roman", Times, serif;
}

Font Style:

p.normal/italic{
font-style: normal/italic;

Font Size

-absolute size:
  • sets the text to a specified size
  • user can't change the text size
  • useful when the size of the output is known

font-size: small;

-relative size:
  • size set relative to surrounding elements
  • user can change the text size

font-size: smaller;

Font Weight

font-weight: normal/bold;
}
